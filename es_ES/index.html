La fantástica biblioteca ORM para Golang

go get -u gorm.io/gorm
  • ORM con todas las funciones
  • Asociaciones (tiene una, tiene muchas, pertenece a, muchos a muchos, polimorfismo, herencia de tabla única)
  • Hooks (antes/después crear/guardar/actualizar/eliminar/encontrar)
  • Carga diligente con precargado, uniones
  • Transacciones, Transacciones anidadas, Punto de guardado, Regresión a punto de guardado
  • Contexto, Modo de declaración preparada, Modo Simulación
  • Insertar en lote, Búsqueda en Lote, Buscar/Crear con Mapa, CRUD con SQL Expr y Contexto Valuer
  • Generador de SQL, Update/Insert, Bloqueos, Optimizador/Indices/Hints, Argumentos Nombrados, Sub-consulta
  • Clave Primaria Compuesta, Indices, Restricciones
  • Migración Automática
  • Registros
  • Extensible, Flexible API: Resolución de Base de Datos (múltiples bases de datos, división de lectura y escritura) Prometheus…
  • Todas las funciones vienen con pruebas
  • Desarrollo Amigable
