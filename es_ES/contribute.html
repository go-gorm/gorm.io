Contribuye a GORM

¡Puedes ayudar a entregar un mejor GORM! hay muchas cosas que puedes hacer:

Ayuda a otros a conocer GORM

  • Comparte GORM con el mundo por canales sociales, o simplemente cuenta a la gente que pueda estar interesada
  • Ayuda a los recién llegados a resolver preguntas en Stack Overflow, Github Problemas y Salas de Chat
  • Escribe tutoriales, graba vídeos, etc… (no olvides añadirlos a la lista)
  • Ayuda a promover GORM en GitHub dándole una estrella y viendo el repositorio GORM

Documentación y redacción

  • Mejorar la documentación de GORM escribiendo nuevo contenido, corrigiendo material existente https://github.com/go-gorm/gorm.io

Traduciendo este sitio

Puedes traducir este sitio actual con https://translate.gorm.io

Para traducir GORM en tu idioma, necesitas publicar una solicitud en Github issues para que sea añadida a https://translate. orm.io.

Programando

  • Crear un pull request en GitHub para solucionar problemas, nuevas características
  • Crear plugins de código abierto para GORM (no olvides añadirlos a la lista)
