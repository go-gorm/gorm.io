A fantástica biblioteca ORM para Golang

go get -u gorm.io/gorm
  • ORM Completo
  • Associações (tem uma, tem muitos, pertence, muitas para muitos, polimorfismo, herança de tabela única)
  • Hooks (antes/depois de criar/salvar/atualizar/delete/encontrar)
  • Eager loading com Preload, Joins
  • Transações, Transações Aninhadas, Save Point, Rollback para Save Point
  • Contexto, Modo de Declaração Preparado, Modo DryRun
  • Inserção em Lote, Localizar/Criar com Map, CRUD com SQL Expr e Context Valuer
  • SQL Builder, Upsert, Locking, Optimizer/Index/Comment Hints, Named Argument, SubQuery
  • Chave Primária Composta, Índices e Restrições
  • Migrações automáticas
  • Logger
  • Plugins API extensíveis e flexíveis: Database Resolver (vários bancos de dados, leitura e escritas separadas) / Prometheus…
  • Todas os recursos desenvolvidos com testes
  • Amigável para Desenvolvedores
