GORM

DokumentasiGenKomunitasAPIKontribusi

Librari ORM yang sangat luar biasa untuk Golang

go get -u gorm.io/gorm
  • Fitur lengkap ORM
  • Gabungan (punya satu, punya banyak, dipunyai oleh, banyak ke banyak, polimorfisme, turunan tabel tunggal)
  • Kaitan (sebelum/sesudah penambahan/simpan/pembaruan/hapus/pencarian)
  • Eager loading dengan Preload, Joins
  • Transaksi, Transaksi Bersarang, Titik Simpan, Kembali ke tanda Titik Simpan
  • Konteks, Mode Pernyataan yang Disiapkan, Mode DryRun
  • Masukan Sekaligus, FindInBatches, Temukan/Buat dengan Map, CRUD dengan SQL Expr dan Context Valuer
  • Pembuat SQL, Upsert, Locking, Optimizer/Index/Comment Hints, Named Argument, SubQuery
  • Perpaduan Primary Key, Indeks, Pembatas
  • Migrasi otomatis
  • Pencatat log
  • Plugin API yang dapat dikembangkan dan fleksibel: Database Resolver (multi-database, pemisahan baca/tulis) / Prometheus…
  • Setiap fitur dihadirkan dengan berbagai tes
  • Memudahkan pengembang
