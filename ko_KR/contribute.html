GORM에 기여하기

더 나은 GORM을 제공하는 데 도움을 줄 수 있습니다! 할 수있는 일이 많이 있습니다

다른 사람들이 GORM을 알도록 도와주세요

  • 소셜 채널을 통해 전 세계와 GORM을 공유하거나 단순히 관심이있을만한 사람들에게 알리십시오
  • 다른 사용자가 Stack Overflow, Github Issues 및 채팅방에서 질문을 해결하도록 도와주세요.
  • 튜토리얼 작성, 영상 녹화 및 기타… ( 이 목록 에 추가해 주세요!)
  • GORM Github 레파지토리에 Star 와 Watching 표시를 하는것은 홍보에 많은 도움이 됩니다.

문서화 및 작성

  • Make GORM’s documentation better by writing new content, correcting existing material https://github.com/go-gorm/gorm.io

사이트 번역하기

https://translate.gorm.io 에서 현제 사이트를 번역하실 수 있습니다

만약 사용하는 언어가 https://translate.gorm.io 에 존재하지 않다면 Github issues 에 남겨주세요

프로그램

  • Github 에서 Pull request를 생성하여 이슈를 고치거나 새 기능을 추가하세요
  • Gorm open-source 플러그인 만들기 (이 목록에 추가해주세요)
