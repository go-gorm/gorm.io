Hooks 생성하기

GORM은 BeforeSave, BeforeCreate, AfterSave, AfterCreate hook을 사용자가 정의 하여 구현 할 수 있습니다. 이러한 hook 메서드는 레코드를 만들때 호출 되며, lifecycle 의 자세한 내용은 Hooks 을 참고 하십시오.

Hooks 메서드를 건너뛰길 원한다면, session mode 에서 SkipHooks 를 사용할 수 있습니다. 예시는 다음과 같습니다.

Map으로 생성하기

GORM은 map[string]interface{} 와 []map[string]interface{}{} 로 부터 create 하는것이 가능합니다.

NOTE map으로 create 할 때, hooks 는 실행 되지 않고, associations은 저장 되지 않으며 primary key 값 도 다시 채워지지 않습니다.

SQL Expression/Context Valuer로 생성

GORM을 사용하면 SQL 표현식을 사용하여 데이터를 삽입 할 수 있습니다.이 목표를 달성하는 데는 두 가지 방법이 있습니다. map [string] interface {} 또는 사용자 정의 데이터 유형에서 생성합니다.
