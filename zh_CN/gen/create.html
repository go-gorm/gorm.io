创建

您可以使用安全类型 创建 方法插入记录，该方法只能在创建数据时接受当前模型的指针。

用选定字段的来创建

您可以在创建数据时使用 Select ，指定插入的字段

忽略字段使用 Omit

批量插入

更高效地插入大量记录，可以将一个 slice 传递给 Create 方法。 将切片数据传递给 Create 方法，GORM 将生成单个 SQL 语句来插入所有数据，并回填主键的值。
